# 概要
複数のpdeファイルを一つにまとめるプログラムです。
まとめた後のpdeファイルが正常に動作することは確認済みです。

# 使い方
1. 自分のスケッチフォルダの直下に PDECollector.jar を入れます。
2. 実行します。

# 生成場所
スケッチフォルダの直下にPDECollectorOutputというフォルダが自動生成されます。
その下に複製されたスケッチフォルダが自動生成されます。
